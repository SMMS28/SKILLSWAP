# Screenshots Directory

This directory will contain screenshots of the SkillSwap application.

## To Add Screenshots:

1. Start the application:
   ```bash
   npm run dev
   npm run client
   ```

2. Run the screenshot helper:
   ```bash
   ./scripts/take-screenshots.sh
   ```

3. Follow the guide in SCREENSHOTS_GUIDE.md

4. Capture screenshots of:
   - Home page
   - Login/Register pages
   - Dashboard
   - Skills page
   - User profiles
   - Exchange details
   - Messages
   - Transactions
   - Mobile views

5. Save images as PNG files with descriptive names
